wkhtmltopdf service





...just make pdf's easy.


Have you ever wondered that there is no easy-peasy web service to make pdf's in the cloud?

And the one or two nice services out there are way to expensive?

Jeah? Just use wkhtmltopdf and relax

__REQUEST__
	GET /api/{{url}}

	{{url}} URL of the website you want converted 
	to PDF as URL encoded String

__RESPONSE__
	Binary PDF Object
